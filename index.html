🏀 Setup

To share match data with all league members, connect a JSONBin.io database. Keys are stored only in your browser.

1

Go to jsonbin.io and create a free account

2

Click API Keys → copy your Master Key

3

Click Bins → New Bin, paste {"matches":[],"aliases":{}} and save

4

Copy the Bin ID from the URL

5

Go to console.anthropic.com → API Keys → create a new key
